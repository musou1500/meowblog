クイックリファレンス

最も使用されるもの

  • --spacing-md - 標準スペーシング (16px)
  • --color-primary - メインブランドカラー
  • --font-size-base - 本文テキスト (16px)
  • --shadow-md - カードシャドウ

決定ルール

  • スペーシング: --spacing-mdから始める
  • テキスト: メインコンテンツには--color-textを使用
  • インタラクティブ: CTAには--color-primaryを使用
  • 立体感: カードには--shadow-mdを使用

よくあるパターン

  • カード: --spacing-md + --shadow-md
  • ボタン: --spacing-sm + --color-primary
  • テキスト階層: base → large → xl
  • スペーシングスケール: xs → sm → md → lg

タイポグラフィ

シンプルなフォントスケール - baseから始めて、見出しは大きく、詳細は小さくします。

目的別選択:

  • ページタイトル: --font-size-xxl (32px)
  • セクション見出し: --font-size-xl (24px)
  • カードタイトル: --font-size-large (18px)
  • 本文テキスト: --font-size-base (16px) ← ここから開始
  • キャプション/メタ: --font-size-small (14px)

スペーシング

ほとんどの場合は--spacing-mdから始めます。密なレイアウトでは小さく、分離には大きくします。

決定ツリー:

  1. ページセクション間？ → --spacing-xxl (48px)
  2. コンポーネント間？ → --spacing-xl (32px)
  3. コンポーネント内？ → --spacing-md (16px) ← デフォルト
  4. ボタンパディング？ → --spacing-sm (8px)
  5. アイコンからテキスト？ → --spacing-xs (4px)

シャドウ

ほとんどのカードには--shadow-mdを使用します。より強調が必要なオーバーレイのみ大きくします。

使用場面:

  • カード、コンテンツブロック: --shadow-md ← ここから開始
  • ドロップダウン、ツールチップ: --shadow-lg
  • モーダル、重要なオーバーレイ: --shadow-xl
  • ホバー効果のみ: --shadow-sm
