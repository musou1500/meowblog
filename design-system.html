Quick Reference

Most Used

  • --spacing-md - Standard spacing (16px)
  • --color-primary - Main brand color
  • --font-size-base - Body text (16px)
  • --shadow-md - Card shadows

Decision Rules

  • Spacing: Start with --spacing-md
  • Text: Use --color-text for main content
  • Interactive: Use --color-primary for CTAs
  • Elevation: Use --shadow-md for cards

Common Patterns

  • Card: --spacing-md + --shadow-md
  • Button: --spacing-sm + --color-primary
  • Text hierarchy: base ‚Üí large ‚Üí xl
  • Spacing scale: xs ‚Üí sm ‚Üí md ‚Üí lg

Typography

Simple font scale - start with base, go bigger for headings, smaller for details.

Choose by Purpose:

  • Page title: --font-size-xxl (32px)
  • Section heading: --font-size-xl (24px)
  • Card title: --font-size-large (18px)
  • Body text: --font-size-base (16px) ‚Üê Start here
  • Caption/meta: --font-size-small (14px)

Spacing

Start with --spacing-md for most cases. Go smaller for tight layouts, bigger for separation.

Decision Tree:

  1. Between page sections? ‚Üí --spacing-xxl (48px)
  2. Between components? ‚Üí --spacing-xl (32px)
  3. Inside components? ‚Üí --spacing-md (16px) ‚Üê Default
  4. Button padding? ‚Üí --spacing-sm (8px)
  5. Icon to text? ‚Üí --spacing-xs (4px)

Shadows

Use --shadow-md for most cards. Only go bigger for overlays that need more emphasis.

When to Use:

  • Cards, content blocks: --shadow-md ‚Üê Start here
  • Dropdowns, tooltips: --shadow-lg
  • Modals, important overlays: --shadow-xl
  • Hover effects only: --shadow-sm
